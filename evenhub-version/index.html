📅 Calendar Companion

Next Event Display for Even Realities G2 Smart Glasses

Even Hub SDK Version

🎯 What is This?

Calendar Companion displays your next upcoming calendar event on your Even G2 smart glasses. It syncs with Google Calendar and automatically updates every 30 seconds, so you always know what's coming up next.

📱 Auto-Sync

Refreshes every 30 seconds

⏰ Time Display

Shows start time & duration

📍 Location

Displays event location

⏳ Countdown

Time until event starts

🎮 Controls

Scroll Down: Open menu
Scroll Up/Down (in menu): Navigate options
Single Tap: Select / Refresh
Double Tap: Open menu / Go back

Menu Options

  • Refresh Now: Manually sync calendar
  • Settings: Configure display preferences
  • View All Events: See upcoming events
  • Toggle Debug: Show/hide debug info
  • Exit: Close the app

🚀 Getting Started

Prerequisites

  • Even Realities G2 smart glasses
  • Even Hub app installed on your phone
  • Google Calendar account
  • Google Cloud Console project with Calendar API enabled

Setup Steps

⚠️ Important: This app requires Google Calendar API credentials. You need to create a Google Cloud Console project and enable the Calendar API.
  1. Create Google Cloud Project:
    • Go to Google Cloud Console
    • Create a new project
    • Enable Google Calendar API
    • Create OAuth 2.0 credentials (Web application)
    • Add authorized JavaScript origins
  2. Configure Environment: 
    # Create .env file
VITE_GOOGLE_CLIENT_ID=your_client_id_here
VITE_GOOGLE_API_KEY=your_api_key_here
  3. Install Dependencies: 
    npm install
  4. Run Development Server: 
    npm run dev
  5. Test in Even Hub Simulator: 
    # Install simulator
npm install -g @evenrealities/evenhub-simulator

# Run simulator
evenhub-simulator
  6. Deploy to Even Hub: 
    # Build production version
npm run build

# Deploy using Even Hub CLI
npm install -g @evenrealities/evenhub-cli
evenhub-cli deploy

⚙️ Technical Details

Display Specifications

  • Resolution: 576×288 pixels (Even Hub SDK coordinates)
  • Actual Hardware: 640×350 pixels
  • Color: Monochrome (1-bit black/white)
  • Image Containers: Max 200×100 pixels each
  • Refresh Rate: 60Hz

Architecture

This app uses the official Even Hub SDK instead of direct BLE communication. It runs as a web application inside the Even Hub app on your phone, which then communicates with your G2 glasses.

✅ Advantages of Even Hub SDK:
  • Official support from Even Realities
  • No BLE UUID discovery needed
  • High-level API for containers and events
  • Easier development and deployment
  • Automatic updates via Even Hub

🐛 Troubleshooting

App Not Loading

  • Make sure you're running inside Even Hub app
  • Check that Google Calendar API credentials are configured
  • Verify internet connection
  • Check browser console for errors

Calendar Not Syncing

  • Verify Google Calendar API is enabled
  • Check OAuth consent screen configuration
  • Ensure authorized JavaScript origins are correct
  • Try signing out and signing in again

Display Issues

  • Text too long? Adjust maxTitleLength in settings
  • Layout broken? Check display composer configuration
  • Events not showing? Verify calendar permissions
